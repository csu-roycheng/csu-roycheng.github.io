48. Rotate Image

You are given an n x n 2D matrix representing an image.

Rotate the image by 90 degrees (clockwise).

Note:

You have to rotate the image in-place, which means you have to modify the input 2D matrix directly. DO NOTallocate another 2D matrix and do the rotation.

Example 2:

给你一个矩阵，将其顺时针旋转90°。注意，你不能使用额外的矩阵。

思路

假设有矩阵如下，该矩阵可以先按照四个角进行旋转
[ 5, 1, 9,11]
[ 2, 4, 8,10]
[13, 3, 6, 7]
[15,14,12,16]

旋转一次后得，下一步旋转之前四个角旁边的数

[ 15, 1, 9, 5]
[ 2, 4, 8,10]
[13, 3, 6, 7]
[16,14,12,11]

经过旋转后得，下一步再旋转旁边的数

[ 15, 13, 9, 5]
[ 2, 4, 8, 1]
[12, 3, 6, 7]
[16,14, 10,11]

经过旋转后得，至此一轮旋转完成，下一轮往里重复上述操作

[ 15, 13, 2, 5]
[ 14, 4, 8, 1]
[12, 3, 6, 9]
[16, 7, 10,11]
